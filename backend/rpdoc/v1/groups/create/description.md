# Description

Create Group.

# Request Header

| Parameters | Description |
| --- | --- |
| **Authorization** | Bearer {{access_token}} |
| **Content-Type** | application/json |

# Request body

| Parameters | Required | Value | Description |
| --- | --- | --- | --- |
| **group_name** | `X` | `String` | Group Name. If not provide, create group with `Default Group Name` |

# Response

See [Group Object](#group-object)
