# General Header Object
| Parameter | Type | Required | Description |
| --------- | ----- | ------ | ----------- |
| Authorization | `String` | `*` | Bearer token, required if member is Kdan member |
| Code | `String` | `*` | JWT token, required if member is not Kdan member. |
| Work-Id | `String` | `*` | An unique ID to identify user, generated by client and required if member is not Kdan member. |
| User-Agent | `String` | `O` | user agent of device or browser |
| Client | `String` | `O` | `web` or `app` |
| Ip-Address | `String` | `O` | ip_address |
| App-Version | `String` | `*` | app version, required if client is app, e.g., `DS_AND_1.1.2` or `iOS_1.1.2` |
